# Elara-symbolic UI

This is a Desmos-like frontend to Elara symbolic and SymPy, allowing using the library without any coding experience. Includes facilities for computing derivatives and integrals (both symbolically and numerically) as well as solving differential equations.

Note that the name right now is TBD, we may keep it as this or switch to another name.

> Currently the app is MIT-licensed, but if all contributors are willing, I would be more than happy to turn it into public-domain licensed software.

## Usage

Assuming you have [Streamlit](https://streamlit.io/) installed, just run:

```bash
python -m streamlit src/main.py
```
